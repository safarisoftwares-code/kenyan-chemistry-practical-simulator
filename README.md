# 🧪 Kenyan Chemistry Practical Simulator

> Interactive KCSE Chemistry Paper 3 simulator for Kenyan secondary school students

**Developed by Safari Softwares, Nairobi, Kenya**

---

## 📖 About

This software simulates KCSE Chemistry Paper 3 practical experiments interactively. Students explore:
- Ion reactions with visual test tube animations
- Flame tests with Bunsen burner simulation
- Acid-base & redox titration with burette readings
- Gas identification tests (animated splint, litmus, bleaching)
- Cation confirmatory tests with NaOH/NH₃
- Complex ion formation simulator
- Enthalpy experiments with temperature-time graphs
- Solubility curves with fractional crystallization
- Practice questions with auto-marking
